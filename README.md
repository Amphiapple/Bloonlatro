# Bloonlatro v0.4.1

A Balatro mod that incorporates Bloons Tower Defense into the base game

Includes 251 Jokers, 15 decks, and more!

# New Additions:

10 towers with full upgrades, along with some changes.

**Warning:** This update is one that will prepare for an upcoming mechanic that will only be implemented in the future. There will be many jokers that shift the balance of the game by bloating the total amount of jokers, which make certain strategies much harder to build into. If you want to play a smaller version of the mod, use the latest update of the previous version, v0.3.4 instead.
